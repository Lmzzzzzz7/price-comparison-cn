# 🛒 中国电商比价助手

基于 OpenClaw 的 AI 比价助手 Prompt，自动搜索淘宝、京东、拼多多等平台，比价并推荐最优选择。

## 功能

- **多平台比价**：淘宝、京东、拼多多、天猫、苏宁易购、抖音商城
- **价格对比**：实时价格 + 运费 + 优惠信息
- **最低价推荐**：自动标注全网最低价
- **历史价格**：查询是否适合入手
- **购物建议**：是否保价、售后服务、店铺信誉

## 使用方法

1. 把 `prompt.md` 里的内容配置到你的 OpenClaw / Claude / GPT 等 AI 助手
2. 告诉 AI 你想买什么，比如"帮我比价 iPhone 15"
3. AI 会自动搜索并返回对比结果
